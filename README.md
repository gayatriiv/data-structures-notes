# data structures and algorithms

notes and code for dsa problems from striver's sheet and neetcode

## topics covered

- c++ basics
- time complexity
- patterns
- basic maths
- hashing
- recursion
- sorting
- arrays
- stl (standard template library)

## structure

```
data-structures-notes/
├── c++ basics/       # fundamentals of c++
├── time complexity/  # big o notation and analysis
├── patterns/         # pattern printing problems
├── basic maths/      # number theory problems
├── hashing/          # hashing techniques using arrays and maps
├── recursion/        # recursive problem solving
├── sorting/          # sorting algorithms
├── arrays/           # array problems
└── stl/              # c++ stl usage
```
